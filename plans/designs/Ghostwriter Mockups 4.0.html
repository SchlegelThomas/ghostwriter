Principles

A craft studio, not a chat beside a doc.

01Prose is sacred

Agents propose; writers apply. No silent overwrite.

02Craft before polish

Sketches, sheets, backdrops help invent — never a second manuscript.

03One scene lease

Draft, Split, Focus, Assist share the same scene ID and save domain.

04Canvas is a sibling

Map stays spatial; Split can show Map/sheet/backdrop beside prose.

05Modalities are equal

Keyboard, dictation, stylus — inspectable text or craft output.

06Keep Tiptap

Canonical scene bodies stay schema-v1 ProseMirror JSON.

07Agents have jobs

Scene Partner, Character Coach, Worldkeeper, Sketch Partner.

08Match live chrome

Write features adopt Map/Draft toolbar density — no parallel UI language.

Feature catalog

What this epic delivers for writers.

F1 · Writing Studio

Draft Desk with Focus, Split, Assist — chrome matches live toolbar density.

Extend
  • Composition switcher as 28px toolbar chips (not a second fat ribbon).
  • Focus clears rail/structure/assist; Esc restores layout.
  • Lease + save state in topbar chips, same as Map “Saved”.

F2 · Scene Sketch

Beat/intent card with the scene; optional Canvas pin.

New
  • Purpose, conflict, turn, sensory notes, open questions.
  • Planned craft — not canon prose.
  • Open from Assist, Canvas card, or Structure menu.

F3 · Character Sheet

Story-knowledge depth for casting a scene.

Extend
  • Desire, pressure, voice, evidence links.
  • Split · Sheet beside Draft.
  • Character Coach proposes sheet updates.

F4 · Backdrop

Place + atmosphere companion; extends scene ambience.

Extend
  • Split shows place notes + mood visual.
  • Canvas can pin the place card.
  • Never auto-inserts setting dumps into prose.

F5 · Writing Agents

Bounded roles; Assist rail uses live panel chrome.

New
  • Scene Partner, Character Coach, Worldkeeper, Sketch Partner.
  • Propose → compare → review → apply → undo.

F6 · Input Modalities

Toolbar toggles for keyboard / dictate / stylus.

Spike
  • Dictation → Tiptap insert at caret.
  • Stylus on sketch/ink craft layer.
  • Optional later: ink-to-text proposal.

F7 · Split compositions

Page · Map · Sheet · Backdrop — one lease.

Extend
  • Split · Map reuses live board surface + card spacing.
  • Opening from Canvas never remounts unrelated scenes.
  • Narrow web stacks panes.

F8 · First-book path

Optional spark → sketch → sheet → first scene.

Phase B
  • “Start writing” always available.
  • Guided / Balanced / Minimal.

Journeys

What the mockup must make obvious.

1 · Sketch → first page

  1. Create scene with book + chapter home.
  2. Fill Scene Sketch.
  3. Optionally pin to Canvas.
  4. Write; sketch stays one click away.

2 · Cast a scene

  1. Pick cast.
  2. Split · Sheet.
  3. Character Coach proposes sharper pressure.
  4. Accept sheet delta; return to prose.

3 · Scene Partner

  1. Assign continue-from-caret.
  2. Two variants vs base revision.
  3. Compare; insert as variant or dismiss.
  4. Working draft unchanged until apply.

4 · Dictate

  1. Toolbar Dictate (⌘M).
  2. Speech inserts at caret.
  3. Stop → keyboard; save stays honest.
  4. Mic denied shows clear recovery.

5 · Stylus sketch

  1. Toolbar Stylus → ink pad.
  2. Draw on craft layer.
  3. Sketch Partner may propose typed fields.
  4. Accept into sketch — not manuscript — unless converted.

6 · Write beside Map

  1. Split · Map or Canvas → Split.
  2. Same scene lease; roomy board spacing.
  3. Rearrange cards; Draft order unchanged.
  4. Focus when the board is noise.
