Intent

Ship the Write epic: Writing Studio chrome, craft artifacts (sketch / character sheet / backdrop), browser dictation, stylus ink on sketches, and bounded writing agents (Scene Partner, Character Coach, Worldkeeper, Sketch Partner) with human apply authority — integrated with Canvas Split and live chrome density.

Locked decisions (D1–D6)

  • D1 Mockups 4.0 accepted as Write living design.
  • D2 Dictation: browser SpeechRecognition first.
  • D3 Ink: craft-only on Scene Sketch (not prose JSON).
  • D4 Agents: ship all four roles; Scene Partner is beachhead UX.
  • D5 Provider: deterministic local assist in v1; optional OpenAI when configured (no silent canon writes).
  • D6 Own branch feat/writing-experience-ux-4 from current main.

Acceptance criteria

  • Write density: Focus, Assist rail, composition chips (Page / Split·Map / Sheet / Backdrop) on live toolbar language.
  • Scene Sketch persists (purpose/conflict/turn/beats/ink) via scene.update; visible in Assist/Split.
  • Character sheet fields (desire/pressure/voice) on character knowledge; Split · Sheet shows cast.
  • Split · Backdrop shows scene ambience companion beside Draft.
  • Dictation inserts into Tiptap at caret with listening chrome.
  • Writing agents return inspectable proposals; apply uses existing commands/variants; dismiss is safe.
  • Stylus ink draws on sketch craft layer and persists with sketch.
  • pnpm verify green; browser walkthrough recorded; Playwright deferred until founder verifies.
