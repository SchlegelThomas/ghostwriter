Provided brand direction

Ghost-writer AI Writing Studio

The supplied lockup is the large-format brand reference for onboarding, splash, and presentation surfaces. Production still needs compact, transparent, monochrome, and app-icon variants; the repository product name remains “Ghostwriter” until naming punctuation is explicitly finalized.

Design posture

  • Warm writing surface, restrained product chrome. Serif typography is reserved for story content; controls remain clear and compact.
  • Four-part desktop shell. Project rail, contextual navigator, central work surface, and collapsible inspector. Focus mode removes everything but the manuscript.
  • The Story Canvas is the visual hero. It replaces Plan and holds scenes, characters, notes, imagery, regions, threads, outline, and timeline as linked views of the same story objects used by Draft.
  • Git safety, writer language. Checkpoints, variants, compare, combine, restore, review, and editions appear as rich content workflows rather than repository mechanics.
  • Human collaboration is first-class. Authors and editors share live project state while comments, tracked suggestions, assistant variants, and approvals remain attributable.
  • AI is visually bounded. Agent surfaces use source receipts, status labels, and proposal containers—not glowing text, avatars, or a chat-first homepage.
  • Authority is readable without color. Canon, planned, inferred, and proposed states always have text labels in addition to color.
  • Narrow web is intentionally smaller. Capture, reading, Inbox decisions, and focused edits replace a compressed four-pane editor while staying in the same shared project.

Narrow web · Focused companion

Capture and proposal review

The responsive web experience prioritizes fast capture and turns the Inbox into one decision at a time. It shows ADR 0002’s minimal reconnect recovery for unacknowledged work, not a complete offline project.

Recovery before enrichment

Capture is durably server-acknowledged before classification, transcription, linking, or AI development. Recovery protects unacknowledged input but does not provide full offline project editing.

Review, not shrink

The phone presents one hunk or finding at a time rather than a tiny desktop diff.

Explicit development

“Develop this” is a separate agent action; saving a note never triggers generation automatically.
