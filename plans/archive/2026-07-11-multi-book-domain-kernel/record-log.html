2026-07-11 13:08 — Scope boundaries recorded

The kernel will define stable aggregate ownership, IDs, ordered manuscript references, scene metadata, edition identity, minimal project-wide story-knowledge references, command/query ports, a memory repository, and capability parity metadata.

It will not implement a database schema, network transport, browser recovery, editor document, revision graph, live collaboration, complete story bible, Story Canvas mutation, import/export, or model call. The client will identify all displayed content as fixture data and will not claim that content is saved or collaborative.
